Configuraciones#

  • Crear una carpeta para el proyecto y dentro ejecutar npm init

  • RECOMENDADO Instalar nodemon: npm install nodemon y agregar al package.json

  • Instalar ExpressJS: npm install express --save

Archivo Inicial#

  • Se crea un archivo index.js o simil:

 1// importamos la librería express:
 2const express = require('express');
 3
 4// instanciamos express:
 5const app = express();
 6// definimos el puerto:
 7const port = 3000;
 8
 9// se preparan los datos hardcodeados:
10let consolas = [
11    {marca: 'Sony', modelo: 'PlayStation', lanzamiento: 1994},
12    {marca: 'Sega', modelo: 'MegaDrive ', lanzamiento: 1991},

14    {marca: 'Microsoft', modelo: 'XBOX', lanzamiento: 2001},
15]
16
17// Creamos una ruta raiz:
18app.get('/', (req, res)=>{
19    // se define la cabezera y el cuerpo:
20    res.status(200).send(consolas); // no hay que parsear a json.

22
23// creamos el servidor:
24app.listen(port, ()=>{

Habilitar CORS#

Por defecto el proyecto tiene las CORS restringidas, de modo que si vamos a conectarnos desde una aplicación externa se bloqueará.

  • Para trabajar con cors se instala npm install cors --save

CORS Publicas#

Habilitar cors para hacer publica la API:

1const express = require('express');
2
3const app = express();
4const port = 3500;
5// importamos la librería cors:
6const cors = require('cors');
7
8// decimos a express que defina los cors (por defecto ahora es pública):
9app.use(cors());

  1const Task = require('../models/taskModel');

  3// la funcion será asíncrona para que espere a terminar de guardar antes de proceder:

 1// importar jwt:

 7// crear token:
 8function createToken(user, expiresIn){

Paso 9: Crear middleware de autenticación#

Para proteger los endpoints, creamos un middleware que verifique si hemos hecho login:

  • Crear una carpeta en la raiz llamada middlewares y dentro el archivo authMiddleware.js:

 1// importamos moment para las fechas:
 2const moment = require('moment');
 3// importamos el servicio que hemos creado con jwt:
 4const jwt = require('../services/jwtService');
 5
 6// copiamos la clave secreta de jwtService.js:
 7const SECRET_KEY = "2ha9df238dhha87d8vaq";
 8
 9// crear una función (next es una función que continuará el proceso si todo va bien):
10function ensureAuth(req, res, next){
11    // si no recibimos el token por el apartado autorización del header:
12    if(!req.headers.authorization){
13        // damos error de autorización:
14        return res.status(403).send({msg: "Faltan las credenciales de autenticación"});
15    }
16
17    // si tenemos la cabecera con el token lo guardamos:
18    const token = req.headers.authorization.replace(/['"]+/g, ""); // reemplazar las comillas simples por nada
19
20    try {
21        // decodificamos el token:
22        const payload = jwt.decodeToken(token, SECRET_KEY);
23
24        // si el token ha caducado respondemos con error:
25        if(payload.exp <= moment().unix()){
26            return res.status(400).send({msg: "el token ha expirado"});
27        }
28
29        // si todo va bien le pasamos el payload al usuario y avanzamos a la función ext:
30        req.user = payload;
31        next();
32    }catch(error){
33        return res.status(404).send({msg: "Token inválido"});
34    }
35}

39    ensureAuth
